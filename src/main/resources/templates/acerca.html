Acerca del Proyecto

Esta aplicación demuestra cómo integrar Spring Boot 4, OAuth2 con autenticación mediante GitHub y Bootstrap 5 para crear una web moderna, segura y responsiva.

¿Qué tecnologías se usan?

  • Spring Boot 4: Framework para crear aplicaciones Java empresariales de manera rápida y sencilla.
  • OAuth2 + GitHub: Autenticación segura usando cuentas de GitHub, sin necesidad de contraseñas propias.
  • Bootstrap 5: Framework CSS para diseño responsivo y componentes modernos.
  • Thymeleaf: Motor de plantillas para integrar el backend Java con vistas HTML dinámicas.

Características principales

  • Login seguro con GitHub usando OAuth2.
  • Gestión de productos (CRUD completo).
  • Control de roles y acceso para usuarios y administradores.
  • Interfaz moderna y responsiva con Bootstrap 5.
  • Uso de fragments para compartir diseño global (navbar, footer, etc.).

¿Para quién es útil?

Ideal para desarrolladores que buscan ejemplos prácticos de integración de autenticación externa, diseño moderno y buenas prácticas en aplicaciones Java web.
